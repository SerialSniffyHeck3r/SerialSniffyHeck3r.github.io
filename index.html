Overview

About this page

This page collects my embedded projects, graduation work, and reverse-engineering writeups. The emphasis is on how things work internally – from signal paths on real boards to weird state machines in old games.

Most of the content below is organized by importance: Core items are the pieces I would point to in an interview, Major are solid supporting work, and Minor are smaller experiments or side notes.

The code lives on GitHub; this page is mainly a structured index and commentary.
